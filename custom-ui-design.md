# レムリアテレビ カスタムUI設計書（Phase 2）

> Phase 1（3/20パイロット）はDaily Prebuilt埋め込みで実施。
> このドキュメントはPhase 2（パイロット後）でのカスタムUI開発用。

---

## 設計思想

- Dailyのビデオ会議ツールに見えないこと
- レムリアテレビの放送を「受信している」体験
- 音声＋チャットが基本。映像はオプション
- 非覚醒メディア：ながら参加、寝落ちOK

---

## 技術スタック

- Daily React SDK（`@daily-co/daily-react`）
- Next.js App Router / TypeScript / Tailwind CSS
- Daily Prebuilt は使わない（フルカスタム描画）

### 必要なDaily React SDKフック
```
useDaily()              → Dailyインスタンス
useLocalParticipant()   → 自分の状態
useParticipantIds()     → 参加者一覧
useAudioTrack()         → 音声トラック制御
useScreenShare()        → 画面共有受信
useDailyEvent('app-message') → チャット送受信
```

### チャット実装
- Dailyの `sendAppMessage()` で送受信
- Daily内蔵チャットではなく自前UIで描画
- デザインの自由度を確保

## /garage（ガレージハント）カスタムUI

### コンセプト
- 音声＋チャットがメイン
- 画面共有はオプション（百面惣が見せたいときだけ）
- チャット欄が主役（声が出せない人の居場所）

### 画面構成（PC）
```
┌──────────────────────────────────────┐
│ [ロゴ小]  GARAGE HUNT                 │ ← ヘッダー
├──────────────────────────────────────┤
│                                      │
│  チャット欄（メインエリア）             │
│  メッセージが流れる                    │
│  半透明背景、レムリアカラー            │
│                                      │
│          ┌──────────┐                │
│          │画面共有    │ ← サムネイル   │
│          │クリックで  │   小さめ表示   │
│          │全画面表示  │                │
│          └──────────┘                │
│                                      │
├──────────────────────────────────────┤
│ 🎤ミュート  🚪退出                    │ ← コントロール
│ 参加者: 百面惣 / ゲスト1 / ゲスト2     │ ← 音声インジケーター
└──────────────────────────────────────┘
```

### 画面共有の全画面表示
- ユーザーが画面共有サムネイルをクリック → オーバーレイで全画面表示
- 「✕ 閉じる」で元のチャットメイン画面に戻る
- 百面惣が画面共有を停止したら自動で閉じる

### 画面構成（モバイル）
```
┌──────────────────┐
│ [ロゴ] GARAGE     │
├──────────────────┤
│                  │
│ チャット欄        │
│ (メインエリア)    │
│                  │
│    [画面共有      │
│     サムネイル]   │
│                  │
├──────────────────┤
│ 🎤  🚪           │
│ 参加者表示        │
└──────────────────┘
```
- インスタライブに近い縦型レイアウト
- チャットが画面の大部分を占める
- 画面共有タップで全画面

### UI要素の詳細

#### チャット欄
- 背景：半透明（rgba(13, 15, 18, 0.7)）
- テキスト色：#E8E4DF
- 送信者名：小さく表示
- 入力欄：下部に固定
- 新しいメッセージは下から追加、自動スクロール

#### 音声インジケーター
- 話している人の名前が光る / パルスする
- シンプルなドット or 波形アニメーション

#### コントロールバー
- 背景：#0D0F12
- ボタン：アイコンのみ、ミニマル
- マイクON = アクセント朱色(#E05A33)のドット
- マイクOFF = グレーアウト

## /nemumi（ねむみチャンネル）カスタムUI

### コンセプト
- レムリアテレビの「本放送」に近い非覚醒チャンネル
- 音声だけ。映像なし
- チャットは背景に溶け込む（Gravity風）
- 眠くなるUI。暗く、静か

### 画面構成
```
┌──────────────────────────────────────┐
│                                      │
│          [ロゴ 大きく中央]             │
│          (新月/チェシャ猫の笑み)       │
│                                      │
│         ねむみチャンネル               │
│      夢の中であいましょう              │
│                                      │
│  ┌────────────────────────────┐      │
│  │ チャット欄                  │      │
│  │ 半透明、背景に溶け込む       │      │
│  │ メッセージがふわっと現れて    │      │
│  │ ゆっくり消える              │      │
│  └────────────────────────────┘      │
│                                      │
│   ねむみ音量 ━━━━━━━●━━━ 70%        │
│                                      │
│           🎤(off)    🚪              │
└──────────────────────────────────────┘
```

### UI要素の詳細

#### ロゴ
- 画面中央、大きめ表示
- 将来的にはSVG化してアニメーション対応
  （呼吸するようにゆっくり明滅する等）

#### チャット（Gravity風）
- 背景：完全透明 or ごく薄い半透明
- メッセージが下から現れて、数秒後にフェードアウト
- 常時表示ではなく「気配」として存在
- 足跡的な機能（「ここにいるよ」が伝わる）
- 入力欄は下部に控えめに

#### ねむみ音量スライダー
- Daily SDKの音声出力ゲインを実際に制御
- スライダーのデザイン：紫系のグラデーション
- ラベル：「ねむみ音量」

#### コントロール
- マイク：デフォルトOFF。アイコンだけ、目立たない
- 退出：小さいアイコン
- カメラ：なし（映像チャンネルではない）

- 背景：#0D0F12

- 日本語：明朝体系（世界観テキスト）
- UI要素：ゴシック体（操作性優先）

### モチーフ
- 閉じた目 / 新月 / チェシャ猫の笑み
- 闇の中の微笑み
- 月相の変化（将来的にステージバリエーション）

## 開発ステップ（Phase 2）

### Step 1：基盤
- `@daily-co/daily-react` 再インストール
- DailyProvider でルーム接続
- 音声の送受信が動くことを確認

### Step 2：チャット
- `sendAppMessage()` でチャット送受信
- 自前UIで描画（ガレージ：常時表示、ねむみ：フェードイン/アウト）

### Step 3：画面共有
- `useScreenShare()` で画面共有を受信
- サムネイル表示 + クリックで全画面

### Step 4：UIの仕上げ
- レムリアのカラーパレット適用
- アニメーション（音声インジケーター、チャットフェード）
- モバイル対応
- ロゴのSVG化・アニメーション

### Step 5：ねむみ音量
- Daily SDKで音声出力のゲイン制御
- スライダーUIと連動
